# Web-Sistem-Informasi-Penjualan-Aan-Cell
A web-based Sales Information System for Aan Cell that manages products, inventory, transactions, and generates daily and monthly reports to improve business efficiency.


## 🏢 Sistem Informasi Penjualan Aan Cell  

### 📌 Gambaran Umum  
Proyek ini merupakan sistem informasi penjualan berbasis web yang dikembangkan untuk membantu pengelolaan usaha Aan Cell. Sistem ini dirancang untuk mempermudah admin dalam mengelola data produk, memantau stok, mencatat transaksi, serta menghasilkan laporan penjualan secara otomatis dan terstruktur.

---

### 🎯 Tujuan  
- Meningkatkan efisiensi pengelolaan data penjualan  
- Mengurangi kesalahan pencatatan manual  
- Mempermudah monitoring stok produk  
- Menyediakan laporan penjualan harian dan bulanan  

---

### 🧩 Fitur Sistem  
- Dashboard admin  
- Manajemen produk (tambah, edit, hapus)  
- Monitoring stok produk  
- Input transaksi penjualan  
- Laporan penjualan (harian & bulanan)  
- Sistem login admin  

---
